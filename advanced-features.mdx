## Advanced Reporting

You access comprehensive analytics and reporting tools to understand your business performance.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
// Get advanced analytics
async function getAnalyticsReport(timeframe) {
  const response = await fetch(`/api/analytics?period=${timeframe}`, {
    headers: { 'Authorization': `Bearer ${apiKey}` }
  });
  return response.json();
}

// Example usage
const analytics = await getAnalyticsReport('last_30_days');
console.log('Revenue by service:', analytics.revenueByService);
console.log('Staff utilization:', analytics.staffUtilization);
```
  ```python
def get_analytics_report(timeframe):
    response = requests.get(
        '/api/analytics',
        params={'period': timeframe},
        headers={'Authorization': f'Bearer {api_key}'}
    )
    return response.json()

# Example usage
analytics = get_analytics_report('last_30_days')
print('Revenue by service:', analytics['revenueByService'])
print('Staff utilization:', analytics['staffUtilization'])
```
</CodeGroup>
