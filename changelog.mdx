## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added AI-powered scheduling recommendations that learn from booking patterns
- Introduced multi-location support for businesses with multiple sites
- Implemented advanced reporting with customizable dashboards
- Added webhook support for real-time booking notifications

## Improvements

- Enhanced mobile responsiveness across all booking interfaces
- Improved API performance with faster response times
- Added bulk operations for managing multiple bookings simultaneously
- Enhanced customer notification system with customizable templates

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Complete API redesign with RESTful endpoints
- Introduced role-based access control for staff and administrators
- Added customer portal for self-service booking management
- Implemented advanced analytics and business intelligence features

## Breaking Changes

- API endpoints updated to v2 structure - migration guide available
- Authentication method changed to API keys instead of basic auth
- Webhook payload format updated with new event structure

</Update>
